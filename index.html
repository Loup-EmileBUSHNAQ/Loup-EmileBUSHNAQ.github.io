Jeu python

J'ai créé un jeu Python dans mon temps libre. Il est loin d'être finit, mais il est aussi bien avancé. Dans ce jeu, deux joueurs s'affrontent. Ils se voient chacun attribués une liste de 5 chiffres, dont la somme est égale. Ils choississent en même temps (ce n'est pas le cas dans mon programme, mais ça devrait l'être) un des 5 chiffres. Le chiffres le plus grand gagne la manche, et les deux joueurs recommencent 5 fois. Le joueur avec le plus de manches remportés gagne la partie. Il y a aussi d'autres modes de jeu, mais ils sont en construction et loin d'être finis. Pour l'instant, seul le mode de base marche. Télécharger le jeu .
